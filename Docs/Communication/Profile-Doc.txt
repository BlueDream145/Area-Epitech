	Requête:

		- Username
        - Name
        - Mail
		- Password
        - Token

		http://127.0.0.1:8080/profileupdaterequestmessage?username={username}&name={name}&mail={mail}&password={password}&token={token}

	Réponse:
		
		- Non nul:

            public ProfileResultEnum Result
